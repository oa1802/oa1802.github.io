During the course of my graduate school studies in Information Technology at Montclair State University, I developed several PHP-based web applications for various classes.

Montclair Eats Grocery Store

Montclair Eats is a fictional grocery store in Montclair, New Jersey. Through the website, groceries can be selected for purchase and a delivery date can be set for when the groceries will be received by the customer. Included in the web application is a login page and different permission levels for general users, employees, managers, and system administrators. The initial stages of building this web application involved defining the database schema before implementation. This website was developed with PHP and MySQL.


Celestial Hotel Reservation System

Celestial is a fictional hotel on Miami Beach. Through the website, hotel rooms can be searched for and booked for particular dates. This website was developed with PHP and MySQL.

Omar's Classic Books is a fictional bookstore in Hoboken, New Jersey. Through the website, books can be selected for purchase and a delivery date can be set for when the books will be received by the customer. This website was developed with JavaScript, Vue.js, PHP, Laravel and MySQL.
